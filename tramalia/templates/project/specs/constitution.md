# Constitution — principios no negociables

1. Spec antes de código: ninguna feature sin tarea en `specs/tasks.md`.
2. Cambios pequeños y verificables.
3. Los gates (`mise.toml`) son obligatorios: el cierre se hace con `tramalia close`.
4. Evidencia antes de cierre: cada tarea deja evidence pack + handoff.
5. La memoria del proyecto vive versionada en el repo (AGENTS.md, docs/ai/).

> Compatible con Spec Kit (`/speckit.constitution`); Tramalia no lo requiere.
